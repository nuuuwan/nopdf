
# nopdf

## Setup and Install

To install a stable version:

```
pip install nopdf-nuuuwan
```

To install a pre-release (which might have more features, but also be
less stable):

```
pip install -i https://test.pypi.org/simple/ nopdf-nuuuwan
```
